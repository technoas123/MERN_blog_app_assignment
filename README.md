# MERN Blog Application

A full-stack blog platform built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to create, edit, manage, and browse blog posts through a modern web interface while demonstrating complete frontend-backend integration.

* * *

## 🚀 Features

-   User-friendly blog creation and management
    
-   Create, edit, update, and delete blog posts
    
-   RESTful API architecture
    
-   MongoDB database integration
    
-   Responsive React-based user interface
    
-   Backend services built with Express.js and Node.js
    
-   Full CRUD (Create, Read, Update, Delete) functionality

## ⚙️ Installation

### Clone the Repository

    git clone https://github.com/technoas123/MERN_blog_app_assignment.git
    cd MERN_blog_app_assignment
    

### Backend Setup

    cd server
    npm install
    npm start
    

### Frontend Setup

    cd client
    npm install
    npm start

## 📸 Application Workflow

1.  Create a blog post
    
2.  Store post data in MongoDB
    
3.  Retrieve posts through REST APIs
    
4.  Display posts on the React frontend
    
5.  Edit or delete existing posts
